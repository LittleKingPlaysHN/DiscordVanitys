## NOTA
- en el .env hay unos datos que debes rellenar asi que los siguientes pasos son importantes.
## COMO HACERLA FUNCIONAR
- DISCORD_CLIENT_ID = aqui pones el id del bot crealo en https://discord.com/developers/applications
- DISCORD_CLIENT_SECRET= = aqui pones el secret ese lo encuentras en https://discord.com/developers/applications/IDDETUBOT/oauth2
- DISCORD_CALLBACK_URL= - aqui pones una url para que rediriga despues que el usuario se logue con discord ejemplo: http://localhost:3000/auth/discord/callback
- SESSION_SECRET= - Aqui pones tu sessionsecret
- DB_HOST= - AQUI LA IP DE LA DB
- DB_USER=- USUARIO DE LA DB
- DB_PORT= - puerto de el enpoint de la DB
- DB_PASSWORD= - aqui la contra de la base de datos
- DB_NAME= - aqui el nombre de la base de datos
- PORT= - puerto de tu hosting
